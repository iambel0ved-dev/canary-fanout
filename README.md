# 🐦 Canary Fanout

Automated content engine for [Canary](https://canary-landing-one.vercel.app) — early warning intelligence for the web.

Canary monitors any URL and alerts you the instant something changes.

## What this does

Automatically detects changes across high-signal websites and distributes intelligence to X, Telegram, LinkedIn, and Substack.

- **Hourly** — detects changes, posts raw facts to X
- **Daily** — generates AI insights + summary, posts thread to X, sends to Telegram
- **Weekly** — deep digest sent to Telegram and email
